# F-Droid Metadata Placeholder

This directory is reserved for draft F-Droid/readiness support files.

It is not active F-Droid submission metadata and does not claim that
pdocker-android is ready for inclusion. Future files here should map to the
release process in `docs/release/FDROID_RELEASE_PROCESS.md` and must keep source,
license, reproducibility, anti-feature, and generated-artifact handling explicit.

`generated-binary-inventory.md` is a support inventory for host-only readiness
checks. It is not active F-Droid build metadata.
